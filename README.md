# 将棋のやり方メモ

## 初期位置と動き
- 駒を初期位置に並べて対局を開始します。
- **動かせる方向、一度に複数マス動かせるかどうか** は、駒によってそれぞれ決まっています。
  - 一度に複数マス動かす際は、相手と自分の駒のどちらも **飛び越えて移動するといった事はできません** 。
  - ただし、「桂馬」だけは例外で、飛び越えての移動が可能です。

## 取る
- 自分の駒が **相手の駒のある場所に乗るような形で「取る」** 事ができます。
  - 逆に言えば **相手の駒の移動範囲にある自分の駒は危険状態** だというイメージです。
  - 取った相手の駒は盤面から一旦取り除き、 **使いたい時に使いたい場所に自分の駒として置ける「持ち駒」** となります。

## 成る
- 自分の駒が **敵陣3列まで侵入できたら裏返して、動きが増えた駒に「成る」** 事が可能です。
  - 多くの駒は「金将」と同じ動きに成ります。

## 王手と詰み
- 最終目的は相手の **王将を取る** 事です。
- 相手の王将を次で取れそうな状態、つまり **自分の駒の移動範囲に相手の王将が入る状態までいったら「王手」** と口で言ってください。
  - 相手側に王手をされてしまったら、 **必ず自分の王将を逃がすなどの対処** をする必要があります。
- 相手を完全に追い詰めて、何をどう動かしても次で必ず王将を取れる状態、つまり **「詰み」** に追い込めば、 **あなたの勝ち** です。
