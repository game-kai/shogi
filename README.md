# 将棋のやり方メモ

## 初めの知識
- まずは駒を初期位置に並べて対局を開始します。
- **駒を動かせる方向、一度に複数マス動かせるかどうか** は、駒によってそれぞれ決まっています。
  - （※動かし方を最初からすべて暗記しようとすると大変です。対局中に確認しながらやってよいです。）
- 一度に複数マス動かせる駒を動かす場合、相手の駒も自分の駒も **飛び越えて移動するといった事はできません** 。
  - ただし、「桂馬」だけは例外で、飛び越えての移動が可能です。

## 取る（攻撃の仕方）
- 自分の駒が **相手の駒がある場所に乗るような形で「取る」** 事ができます。
  - 逆に言えば、 **相手の駒の移動範囲にいる自分の駒は、取られる危険性がある状態** だというイメージをしてください。
- 取った相手の駒は、盤面から一旦取り除き、 **使いたい時に使いたい場所に自分の駒として置ける「持ち駒」** となります。
  - 持ち駒を置いた時は、駒を動かす事はできず、そこで番が終わります。
  - （※「歩兵」だけは、複数個、縦には存在できない「二歩」という反則が存在するので、置くときに注意しましょう。）
- 自分の駒を取る事などはもちろんできません。
  - その為、自分の駒が退路を断つ壁となってしまう事がありえます。

## 成る（駒を強くする方法）
- 自分の駒が奥の **敵陣3列まで侵入できた場合、裏返して、動ける方向が増えた駒に「成る」** 事が可能です。
  - （※成った後の動きは「金将」と同じである物がほとんどですが、これについても確認しながらやってよいです。）
  - 駒を成らせるか成らせないかは、その都度判断して決める事ができます。
  - 持ち駒を敵陣に直接置いた直後は成る事ができません。1回でもその駒を動かしたら成る事ができます。

## 王手と詰み（クライマックス）
- 最終目的は相手の **王将を追い詰める** 事です。
  - 盤面がどうであれ、持ち駒の状態がどうであれ、どちらかが相手の王将を追い詰めたらそこで対局は終了です。
- 相手の王将を、このままなら **次の自分の番に取る事ができる状態となったら「王手」** と口で言ってください。
  - つまり、 **自分の駒の移動範囲に相手の王将が入った時に宣言をする** イメージです。
  - 相手側に王手をされてしまったら、必ず以下のような対処をする必要があります。
    - 自分の王将を **逃がす**
    - 王手をしている相手の駒を **取る**
    - 自分の駒を間に隔てる事で **守る**
  - ここでこれら以外の関係ない行動を取ると、当然負けとなってしまいます。
- 相手を完全に追い詰めて、何をどう動かしても次で必ず王将を取れる状態、つまり **「詰み」** に追い込めば、 **あなたの勝ち** です。
