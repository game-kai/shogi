# 将棋のやり方メモ

## 初期位置と動き
- まずは駒を初期位置の陣形に並べて対局を開始します。
- **駒を動かせる方向、一度に複数マス動かせるかどうか** は、駒によってそれぞれ決まっています。
  - 一度に複数マス動かせる駒を動かす場合、相手の駒も自分の駒も **飛び越えて移動するといった事はできません** 。
  - ただし、「桂馬」だけは例外で、飛び越えての移動が可能です。

## 取る
- 自分の駒が **相手の駒のある場所に乗るような形で「取る」** 事ができます。
  - 取った相手の駒は盤面から一旦取り除き、 **使いたい時に使いたい場所に自分の駒として置ける「持ち駒」** となります。

## 成る
- 自分の駒が奥の **敵陣3列まで侵入できた場合、裏返して、動きが増えた駒に「成る」** 事が可能です。

## 王手と詰み
- 最終目的は相手の **王将を取る** 事です。
- 相手の王将をこのままなら **次の自分の番に取る事ができる状態までいったら「王手」** と口で言ってください。
  - 相手側に王手をされてしまったら、必ず自分の王将を逃がすなどの対処をする必要があります。
- 相手を完全に追い詰めて、何をどう動かしても次で必ず王将を取れる状態、つまり **「詰み」** に追い込めば、 **あなたの勝ち** です。
